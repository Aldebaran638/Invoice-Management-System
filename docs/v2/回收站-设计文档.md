# 回收站设计文档

工具组：购买记录  
工具组英文标识：purchase-records

系统规则：  
1. 回收站只展示逻辑删除的购买记录。  
2. 回收站中的购买记录最多保留30天。  
3. 30天起算时间必须为 `deleted_at`。  
4. 超过30天的购买记录必须由后台定时任务自动清除。  
5. 查询接口禁止顺手清理超时记录。
