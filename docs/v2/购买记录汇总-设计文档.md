工具组：购买记录
工具组英文标识：purchase-records
工具名称：购买记录汇总
工具英文标识：purchase-record-summary

1. 购买记录汇总只提供列表查询与详情查询。
2. 页面展示字段只允许包含购买日期、名称、购买金额。
3. 管理员可查看全部购买记录。
4. 普通用户只可查看自己的购买记录。
5. 本工具复用现有购买记录表，不新增表，不扩展字段。
