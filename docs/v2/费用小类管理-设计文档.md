工具组：系统管理
工具组英文标识：system-management
工具名称：费用小类管理
工具英文标识：expense-subcategory

1. 本工具为新建工具，属于系统管理组。
2. 仅管理员可操作，普通用户无权限访问。
3. 费用小类初始数据：自动导航承载车、智能喷漆机器人、钢筋折弯与结扎机器人、生产线车队调度、研发部开销。
4. 小类必须关联一个大类(major_category_id)。
5. 小类名称不可重复。
6. 删除小类前必须确认无关联的购买记录。
